== Gallery manager
{Link}[http://http://stackoverflow.com/questions/2230954/is-there-any-way-to-easily-link-a-file-in-rdoc] to demo on heroku

Easy app to upload photo and generate json with links.

Photos are upload to amazoon s3. We get the json file with url for thumbnails and normal size photo. 
We can very fast connect this app with easy gallery on any website.
